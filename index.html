About Me

Saya merupakan lulusan S1 Informatika, Institut Teknologi Nasional Bandung tahun 2023. Karena hasil kerja keras dan ketekunan, saya juga berhasil mendapat predikat cumlaude dengan perolehan IPK 3.64. Saya sering mengerjakan proyek mandiri yang berkaitan dengan pembangunan aplikasi berbasis web, data mining, dan robotika.

Kelebihan : Salah satu kelebihan saya adalah cepat untuk menganalisis suatu permasalahan dan segera menemukan solusi terbaiknya. Saya pernah diberikan proyek untuk mengerjakan pembangunan aplikasi berbasis web dengan deadline yang cukup singkat. Tapi hal tersebut tidak menjadi masalah, karena saya menganggap hal tersebut sebagai tantangan, dan proyek tesebut tetap bisa berfungsi secara maksimal. Salah satu kebiasaan sebelum saya bekerja adalah menganalisis permasalahan yang terjadi lalu mengorganisir seluruh dokumen dan tools yang saya butuhkan untuk menyelesaikan masalah tersebut.
Kelemahan : Saya sering merasa kurang percaya diri dan kerap merasa gugup saat harus presentasi atau berbicara di depan umum. Untuk mengatasinya, saya biasanya mempersiapkan diri dan juga materi sehari sebelum presentasi.
